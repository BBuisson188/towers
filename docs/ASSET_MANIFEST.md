## Production note

These PNGs are concept references, not game-ready 3D meshes, textures, animation rigs, UI atlases, or legally cleared final production assets. For the first prototype, use original low-detail placeholder models. Later, create original 3D assets inspired by the approved silhouettes and palette rather than attempting to extract production assets from the images.
